## 감정 설계 원칙 (P1)

### 3대 감정 목표
1. **따뜻함** — 크림톤 배경, 피치 액센트, 둥근 형태
2. **편안함** — 압박 없는 문구, 부드러운 그림자, 넉넉한 여백
3. **즐거움** — 귀여운 아이콘, 칭찬 마이크로카피, 작은 보상감

### 색채심리 가이드
- 민트(#4fd1c5): 안정, 치유 → 주요 액센트 유지
- 크림(#fdfcfa): 따뜻한 표면 → 카드 배경에 활용
- 피치(#ffab91): 사랑, 온기 → 강조 포인트
- 라벤더(#c4b5fd): 편안, 이완 → 약/건강 관련
- 소프트 그린(#d6f5f1): 자연, 생명 → 배경 그라데이션

## 변경 목록

### 1. 로그인 화면 고양이 아이콘
**문제**: 기하학적 도형 조합. 차갑고 귀엽지 않음.
**해결**: 카와이 스타일 SVG. 둥근 얼굴, 볼터치, 반짝이 눈, 작은 하트.
**파일**: staging/components/auth/LoginScreen.tsx (WelcomeCatIcon 교체)

### 2. 스플래시 로고 흰배경
**문제**: logo.jpeg 흰 배경이 민트 그라데이션에서 사각형으로 보임.
**해결**: border-radius + overflow:hidden + box-shadow로 자연스럽게 블렌딩.
**파일**: staging/components/shell/ShellSplashGate.module.css

### 3. 방송 케어버튼
**문제**: 4가지 다른 색 그라데이션 → 산만, 무지개 떡.
**해결**: 통일된 소프트 카드 스타일. 흰 배경 + 색상 아이콘뱃지 + 따뜻한 카피.
**파일**: staging/components/broadcast/CameraBroadcastClient.module.css

### 4. 커뮤니티 페이지
**문제**: 기본 CRUD. 돌봄 앱 특유의 기능/감성 없음.
**해결**: 돌봄일지, 건강상담, 사진자랑, 꿀팁 카테고리. 아이콘+색상 차별화.
**파일**: staging/components/community/CategoryCard.tsx, Community.module.css

### 5. 홈 CareStatusGrid
**문제**: 기계적 문구, 차가운 톤.
**해결**: 따뜻한 카피, 소프트 아이콘 색상, 격려 마이크로카피.
**파일**: staging/components/home/CareStatusGrid.tsx
