# Claude Design 프롬프트 — 홈 대시보드

## 아래 텍스트를 Claude Design에 그대로 복사해서 붙여넣으세요

---

CATvisor (다보냥) 모바일 웹앱의 홈 대시보드 디자인을 만들어주세요.

**브랜드 정체성:**
- 이름: 다보냥 (CATvisor)
- 한 줄 정의: "귀여운 감시자가 아니라, 같이 살아가는 가족"
- 사용자: 30-40대 집사, 특히 아픈 고양이 보호자, 초보 집사, 출근하는 1인 가구
- 핵심 감정: "혼자가 아니에요" — 외로움 해소, 따뜻한 신뢰감

**디자인 방향:**
- Warm Minimal Korean (토스 같은 신뢰감 + 펫 브랜드의 따뜻함)
- 의료 앱처럼 단정하지만, 카피로 감성 전달
- AI 슬롭 금지: 보라 그라데이션 X, 아이콘 3열 그리드 X, 중앙 정렬 일색 X

**색상 팔레트:**
- Primary: 민트 #4fd1c5 / 민트 강조 #38bdb0
- 배경: 연한 민트 #e0f7fa
- 카드: 흰색 rgba(255,255,255,0.92)
- 경고 스펙트럼 (상태별):
  - 안심: 세이지 #a8d5ba
  - 주의: 버터 옐로 #ffd699
  - 경고: 코랄 #ff8a65
  - 심각: 빨강 #ff5252
- 커뮤니티 accent: 라벤더 #a29bfe

**타이포그래피:**
- 본문 + UI: Pretendard Variable (한국 웹 앱 표준)
- 고양이 이름만 Instrument Serif (예: "보리", "찹쌀이" — Serif로 주인공 강조)
- 숫자: tabular-nums

**레이아웃:**
- 모바일 세로 비율 (9:16 또는 iPhone 14 Pro mockup)
- 배경은 연한 민트
- rounded-3xl 카드 (border-radius 24px)
- 부드러운 민트 기반 그림자

---

**화면 구성 (상단부터):**

### 1. 상단 헤더 (고정)
- 왼쪽: "다보냥" 로고 (고양이 아이콘 + Pretendard Bold)
- 오른쪽: 알림 벨 아이콘 (빨간 점 배지 1개 표시, 흔들림 힌트)

### 2. Hero Card — AI 상태 요약 (가장 먼저 보이는 영역)
흰 카드 안에:
- 큰 이모지 😴
- 메인 문구: "**보리**와 **찹쌀이** 편하게 쉬고 있어요"
  - (보리, 찹쌀이는 Instrument Serif로 강조, 나머지는 Pretendard)
- 서브: "마지막 식사 3시간 전 · 이상 없음" (회색 톤)
- 그 아래 카메라 썸네일 2개 (가로 2열):
  - 각각 LIVE 빨간 점 배지
  - 카메라 피드 이미지 (고양이 누워있는 모습)
  - 좌하단에 고양이 이름 ("보리", "찹쌀이" Serif)

### 3. 오늘의 돌봄 — 보호자 할 일 체크리스트
섹션 제목: "오늘의 돌봄"
2x2 그리드로 4개 카드:

**카드 1 — 화장실 청소**
- 아이콘: ✨ (또는 Lucide Sparkles)
- 메인 문구: "반짝반짝 깨끗! ✨"
- 서브: "1일 전"
- 배경: 흰색, 좌측 얇은 세이지 accent

**카드 2 — 물 갈아주기**
- 아이콘: 💧 (또는 Lucide Droplets)
- 메인 문구: "깨끗한 물 OK! 💧"
- 서브: "1일 전"

**카드 3 — 밥 주기** (⚠️ "밥 먹기"가 아니라 "밥 주기")
- 아이콘: 🍚
- 메인 문구: "아직 식사 전이에요 🍚"
- 서브: "오늘 첫 끼를 기다리는 중"

**카드 4 — 약 챙기기**
- 아이콘: 💊
- 메인 문구: "약 잘 먹었어요 💊"
- 서브: "2일 전"

### 4. 우리집 실시간 — 카메라 섹션
(만약 위 Hero에서 카메라를 안 보여줬다면 여기 큰 카메라 피드 1-2개)

### 5. 친구들 미리보기
흰 카드 (좌측 라벤더 accent):
- 아이콘: 💬 말풍선
- "같은 고민 하는 집사님들"
- "🐱 보리 같은 심장 이상 가진 아이 보호자 **52명**이 대화 중이에요"
- "대화 참여하기 →" 버튼

### 6. 오늘 있었던 일 (최근 알림)
섹션 제목: "오늘 있었던 일" (기존 "RECENT ALERTS"가 아님)
리스트 카드:
- 🐾 "찹쌀가 그루밍 중이에요 ✨"
  - 시간: "3월 30일 오전 10:58"
- 🐾 "보리가 그루밍 중이에요 ✨"
  - 시간: "3월 30일 오전 10:58"

### 7. 하단 탭바 (고정)
4개 탭 (한국어 통일):
- 홈 (House 아이콘, 활성 상태 - 민트)
- 리포트 (BarChart 아이콘)
- 친구들 (Users 아이콘)
- 설정 (Settings 아이콘)

**톤 & 디테일:**
- 숫자는 Pretendard Semibold, tabular-nums
- 고양이 이름 (보리, 찹쌀이)은 항상 Instrument Serif
- 섹션 간 간격 충분히 (24-32px)
- 카드 모서리는 rounded-3xl
- 이모지는 남발 금지 (문장당 1개)
- 영어 UI 라벨 금지 (모든 라벨 한국어)

**출력:** 모바일 세로 스크린샷 (375x812 기준)
